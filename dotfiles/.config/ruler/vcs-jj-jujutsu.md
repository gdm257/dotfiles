Use Jujutsu(jj) Version Control System if one of below conditions occur:

- `.jj` directory exists.
- `.git` directory does not exist.
- `gh` command fails to load repo info.
- VCS is needed.

Learn more about `jj` command:

- Run `jj help` for docs.
- E.g. `jj help`, `jj help bookmark`, `jj help bookmark list` etc.
